# Data Cleaning & Preparation Project (Week 1)

## Project Overview
Cleaned and prepared a raw corporate dataset containing 1,100+ records using Excel to make it analysis-ready.

## Fixes Performed
- **Missing Values:** Identified blank cells across columns and filled them with standard 'N/A' to prevent downstream calculation errors.
- **Data Deduplication:** Removed redundant and repeat entries to maintain data integrity.
- **Formatting & Logic:** Standardized inconsistent text casing and stripped out trailing spaces using dynamic functions.
